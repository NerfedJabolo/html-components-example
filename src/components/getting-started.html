Getting Started

What is HTML Components?

HTML Components is a lightweight and versatile JavaScript solution for including and rendering HTML components dynamically in web pages. It allows you to modularize your HTML code into reusable components and load them asynchronously at runtime. With its minimalistic approach and powerful features, HTML Components simplifies the development and maintenance of complex web applications.

Prerequisites

The guide expects you to have a basic understanding of HTML. The guide will assume that you are familiar with HTML tags, attributes, and syntax. If you are not familiar with HTML, you can learn it from W3Schools.

In this guide we use a src folder with the script and an index.html file and a components folder where we will make the components. Note that this setup is not mandatory and you can name the files and folders as you like.
